# [AndroidTrainingPlan](https://17media.atlassian.net/wiki/spaces/InStream/pages/706478430/Android+Training+Plan)

## [Android]Training project - image search application

 
### 專案需求：

- 具有調整顯示方格或直列的樣式，並紀錄樣式設定，下次啟動為最後設定樣式

- 可以下拉刷新，上滑加載更多

- 具有搜尋與歷史紀錄功能

- 框架使用MVVM

- DI使用Koin

- 網路使用Retrofit, OkHttp,

- 資料庫使用Room
